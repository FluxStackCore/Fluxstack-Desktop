# Fontes de Download do Chromium

## 1. Chrome for Testing (Atual - Google Oficial)
- URL: https://googlechromelabs.github.io/chrome-for-testing/
- Prós: Estável, API oficial, bem documentado
- Contras: Aviso "for testing", maior

## 2. Chromium Snapshots (Google Storage)
- URL: https://www.chromium.org/getting-involved/download-chromium/
- Exemplo: https://commondatastorage.googleapis.com/chromium-browser-snapshots/index.html
- Prós: Chromium puro, open source
- Contras: Snapshots podem ser removidos, sem API garantida

## 3. Playwright Chromium Builds
- Playwright mantém: https://playwright.azureedge.net/
- Usado por: Playwright, muito testado para automação
- Prós: Otimizado para automação, estável
- Contras: Versionamento específico do Playwright

## 4. Chromium Builds Oficiais (Continuous)
- https://download-chromium.appspot.com/
- Builds contínuas do projeto Chromium
- Prós: Sempre atualizados
- Contras: Menos estável

## Recomendação Atual
Chrome for Testing pela estabilidade, mas podemos mudar para
Chromium puro se o aviso "for testing" for um problema.
