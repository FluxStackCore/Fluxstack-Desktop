# FluxDesktop Roadmap

> **Note** |
> FluxDesktop is now part of the FluxStack ecosystem! For questions and contributions, visit the [FluxStack repository](https://github.com/MarcosBrendonDePaula/FluxStack).

## v1.1.0 - Q1 2024
- [ ] Enhanced FluxStack integration with automatic detection
- [ ] Built-in FluxStack dev server bridge
- [ ] Hot reload sync with FluxStack apps
- [ ] System tray API for desktop notifications
- [ ] Auto-updater integration
- [ ] Package manager for FluxDesktop plugins

## v1.0.0 ✅ - November 2024
- [X] Complete TypeScript rewrite with Bun runtime
- [X] Native TypeScript development (no build step)
- [X] Updated API from `window.Gluon` to `window.FluxDesktop`
- [X] FluxStack ecosystem integration
- [X] Type-safe IPC communication
- [X] Hot reload development experience
- [X] Modern documentation and examples

## Future Ideas
- [ ] Cross-platform packaging (Windows, macOS, Linux)
- [ ] Electron migration helper for existing apps
- [ ] Plugin system for custom desktop features
- [ ] Performance monitoring dashboard
- [ ] Multi-window management
- [ ] Native OS integration (file associations, protocol handlers)
- [ ] Auto-deployment to app stores

## Legacy (Gluon Era) ✅
- [X] Browser detection system
- [X] Chrome DevTools Protocol integration
